# מערכת תורים למספרת כלבים - צד לקוח

פרויקט React + TypeScript עבור מערכת ניהול תורים למספרת כלבים.

## טכנולוגיות

- React
- TypeScript
- Material UI
- Axios

## יכולות

- התחברות למערכת
- הרשמת משתמש חדש
- צפייה בתורים
- הוספת תור
- עריכת תור
- מחיקת תור
- סינון לפי שם לקוח
- סינון לפי תאריכים
- הצגת פרטי תור בחלון קופץ

## הרצת הפרויקט

```bash
npm install
npm run dev
```
